Welcome to Akosix

Akosix is a free and open-source hobby operating system for the x86 computers (for Intel, AMD, VIA 32 processor).

Gathering and building Akosix

$ sudo apt-get install git git-core build-essential xorriso qemu # Qemu is optional
$ git clone git://github.com/akoskovacs/Akosix.git # Download the repo
$ cd Akosix
$ make        # Build
$ make iso    # Create .iso [optional]
$ make qemu   # Test in Qemu

Features

  • Console driver
  • Partial virtual memory management with paging
  • Multiboot compatibility

Goals

  • Be usable
  • Some user-space application
  • Minimal Linux binary compatibility (this will be the default for Akosix apps)
  • Minimal MS-DOS compatibility
  • Simple package management system
  • Some drivers
    • Filesystems
      • Iso (cd9660)
      • Fat32
      • Ext2 or something similar
    • Networking
      • Small TCP/IP stack
      • 3com or Realtek ethernet
    • PCI bus
    • ATA disk and cdrom
